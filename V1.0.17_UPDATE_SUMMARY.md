# Version 1.0.17 Update Summary
**Date:** November 18, 2025
**Build:** 21

## Critical Bug Fixes

### 1. Daily Goals Sync Issue Fixed
**Problem:** Goals calculated during onboarding weren't displaying on homepage
**Root Cause:** App reading from local SharedPreferences instead of Supabase database
**Solution:** Switched to SupabaseUserProvider as single source of truth

**Files Modified:**
- `lib/screens/onboarding/supabase_onboarding_screen.dart` - Now reloads from Supabase
- `lib/screens/main/nutrition_home_screen.dart` - Updated Consumers
- `lib/screens/main/progress_screen_new.dart` - Updated Consumers

**Impact:** Goals persist correctly from onboarding to homepage ✅

## Major UX Improvements

### 2. Nutrition Display Enhancement
**User Request:** Show actual consumption vs goal, not just remaining
**Changes:**
- **Calorie Display:** "250 KCAL LEFT" → "1950 / 2200 KCAL"
- **Macro Displays:** "30g left" → "120 / 150g"

**Benefits:**
- Reduced cognitive load (no mental math)
- Better decision making
- Clearer progress tracking

**File Modified:** `lib/screens/main/nutrition_home_screen.dart` (lines 520, 528, 684-704)
